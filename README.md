# AirFire

**Private peer-to-peer dogfighting game.** Fly small airplanes with friends and battle in the sky.

## Current Status
- Cross-platform (works on phones, tablets, and desktops)
- Shared via URL (host acts as the server using NetplayJS rollback netcode)
- Modern 80s aesthetic (in progress)

## Project Structure
- `index.html` — Single-file game (lobby + game)
- Built with NetplayJS for multiplayer

## Development Notes
- We are currently prioritizing:
  1. Gameplay / Flow / Responsiveness
  2. UI / Layout / Art (Modern 80s theme)

## How to Run
Simply open `index.html` in a browser. For multiplayer, one person hosts and shares the link.

## Goals
- Keep the game simple and fun with friends
- Excellent responsiveness across all devices
- Strong "Modern 80s" visual identity
- Smooth, responsive controls (especially on mobile)
